# About
## A simple todo list app built with Vue 2

### Require Stack: Node.js `v16.17.0`, Yarn `v1.22.19`, Vue CLI `5.0.8`

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
